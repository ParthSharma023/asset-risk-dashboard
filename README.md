# Asset Risk Dashboard


Interactive Dash app for exploring how maintenance strategies affect asset risk

and lifecycle cost over time.


## Overview

- Models risk as Likelihood of Failure (LOF) Ã— Consequence of Failure (COF).

- Compares four strategies: No Fix, Fix in Plan, Fix on Fail, and Fix on Risk.

- Visualizes risk curves, cumulative cost, trade-offs, and a summary dashboard.
